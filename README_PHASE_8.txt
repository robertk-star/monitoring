SAFFHIRE MONITORING - PHASE 8 MONITORING AUTOMATION

What this phase adds:
- Monitoring Alerts panel on the Monitoring page
- Counts and quick filters for:
  - Total
  - On Monitoring
  - Off Monitoring
  - Expired Medical
  - Expiring in 30 Days
  - Expiring in 60 Days
  - Missing Medical
  - MVR Attention
- Medical card warning badges in each row
- Row warning stripes
- Copy Summary button
- Download Current View CSV button
- Copy Row button for each applicant

Files included:
- index.html
- public/phase8.js
- README_PHASE_8.txt

SQL needed:
No.

Vercel ENV needed:
No.

Install:
1. Upload these files over the existing project.
2. Redeploy Vercel.
3. Go to Monitoring.

What to test:
1. Confirm the Phase 8 Monitoring Alerts panel appears.
2. Click Expired Medical.
3. Click Expiring 30 Days.
4. Click Missing Medical.
5. Click Copy Summary.
6. Click Download Current View CSV.
7. Confirm row badges/stripes appear where medical cards are expired or expiring.
